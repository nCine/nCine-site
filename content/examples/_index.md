This page lists most of the nCine application tests (or AppTests). They live in the [`tests`](https://github.com/nCine/nCine/tree/master/tests) and [`src/tests`](https://github.com/nCine/nCine/tree/master/src/tests) directories of the repository. They are used to stress-test the API and show users how to use it.

Most of them are available on the web using [Emscripten](https://emscripten.org/).
